# Task Tracking

Use the files in this folder for work that should be tracked across agent sessions.

## Rules

- Keep each task as a Markdown checkbox: `[ ]` for open, `[x]` for complete.
- When finishing a task, mark it complete in the relevant file instead of leaving completion in chat only.
- Add a short completion note with the date and changed files under the task file's `## Notes` section.
- Do not delete incomplete tasks. If scope changes, rewrite the task so the remaining work is still explicit.
- If new work is discovered, add new unchecked items to the most relevant task file.
